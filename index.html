Finbot

Aplicación web interactiva que simula un asistente tipo chat enfocado en experiencias guiadas mediante botones y flujos estructurados.

A diferencia de un chatbot tradicional basado solo en texto, Finbot prioriza la usabilidad mediante opciones claras que reducen la fricción del usuario y mejoran la toma de decisiones.

Este proyecto fue desarrollado como un MVP funcional para validar la interacción usuario → bot → respuesta, implementando lógica básica de flujo y respuestas dinámicas en un entorno web.
